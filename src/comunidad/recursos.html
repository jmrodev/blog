Repositorios recomendados para personalización avanzada

  • base46: Temas y utilidades de color para personalizar la apariencia de NvChad.
  • nvcommunity: Plugins y módulos de la comunidad para ampliar funcionalidades.
  • starter: Plantilla base oficial para empezar con NvChad.
  • tinyvim: Configuración minimalista para usuarios avanzados.
  • ui: Elementos visuales y utilidades para mejorar la interfaz.
