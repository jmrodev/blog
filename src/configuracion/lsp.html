LSP y Mason

¿Qué es LSP?

El Language Server Protocol (LSP) permite autocompletado, saltos, acciones de código y más, integrando herramientas inteligentes para cada lenguaje.

¿Qué es Mason?

Mason es un gestor de servidores LSP, linters y formatters. Permite instalar y actualizar herramientas fácilmente desde Neovim.

Instalar un servidor LSP

  1. Abre :Mason y busca el servidor que necesitas (ej: pyright para Python).
  2. Instálalo desde la interfaz de Mason.
  3. Agrega el nombre del servidor en nvim/lua/configs/lspconfig.lua: 
    local servers = { "html", "cssls", "pyright" }
  4. Reinicia Neovim o ejecuta :LspRestart.

Ejemplo de configuración para varios lenguajes

local servers = { "html", "cssls", "tsserver", "pyright", "clangd" }

Comandos útiles y troubleshooting

  • Usa :LspInfo para ver el estado de los servidores LSP.
  • Si algo no funciona, revisa el nombre exacto del servidor y que esté instalado.
  • Reinicia Neovim o ejecuta :LspRestart tras cambios en la configuración.
