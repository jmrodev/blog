Gestión de plugins (Lazy.nvim)

¿Qué es un plugin?

Un plugin es una extensión que añade funcionalidades a Neovim, como autocompletado, explorador de archivos, temas, etc.

¿Qué es Lazy.nvim?

Es el gestor de plugins moderno de NvChad. Permite instalar, actualizar y eliminar plugins de forma visual y sencilla.

Comandos útiles

  • :Lazy → Abre la interfaz visual de Lazy
  • :Lazy sync → Instala/actualiza/elimina plugins según tu configuración
  • :Lazy update → Actualiza todos los plugins
  • :Lazy clean → Elimina plugins no usados

¿Cómo agregar un plugin?

  1. Edita nvim/lua/plugins/init.lua y añade una tabla con el plugin. Ejemplo:
    {
  "ggandor/lightspeed.nvim",
  event = "VimEnter",
},
  2. Guarda el archivo y ejecuta :Lazy sync en Neovim.

¿Cómo eliminar o desactivar un plugin?

  • Elimina la tabla correspondiente en init.lua y ejecuta :Lazy sync.

Tips

  • Lee la documentación de cada plugin para configuraciones avanzadas.
  • Usa eventos como event = 'VeryLazy' para optimizar el rendimiento.
  • Haz cambios de uno en uno y prueba que todo funcione.

Ejemplo avanzado de plugin

return {
  {
    "max397574/better-escape.nvim",
    event = "InsertEnter",
    config = function()
      require("better_escape").setup()
    end,
  },
  -- ...otros plugins
}

Consejos de rendimiento y troubleshooting

  • Usa event = 'VeryLazy' o event = 'InsertEnter' para cargar plugins solo cuando se necesiten.
  • Si un plugin no funciona, ejecuta :Lazy sync y revisa la configuración.
  • Consulta la documentación de cada plugin para opciones avanzadas y solución de problemas.

Plugins de la comunidad (nvcommunity)

  • El repositorio nvcommunity contiene plugins y módulos desarrollados por la comunidad para ampliar las capacidades de NvChad.
  • Puedes explorar repos/nvcommunity/lua/nvcommunity/ para encontrar integraciones de autocompletado, LSP, edición, exploradores de archivos y más.
  • Para usar un módulo, sigue las instrucciones del README.md correspondiente y añade la configuración sugerida en tu plugins/init.lua.
  • Verifica la compatibilidad de cada módulo con tu versión de Neovim y otros plugins antes de integrarlo.
