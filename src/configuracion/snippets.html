Snippets (LuaSnip)

¿Qué es un snippet?

Un snippet es un fragmento de código reutilizable que puedes insertar rápidamente para ahorrar tiempo y evitar errores.

¿Cómo usar LuaSnip?

  • Pulsa Tab para expandir un snippet o saltar al siguiente campo.
  • Pulsa Shift + Tab para volver al campo anterior.
  • Pulsa Ctrl + l para elegir una opción en un campo de elección.

¿Cómo crear y cargar snippets?

  1. Crea un archivo en nvim/lua/snippets/ para tu lenguaje.
  2. Ejemplo para Lua:
    return {
  s("fn", {
    t("function "), i(1, "nombre"), t("()\n  "), i(0), t("\nend")
  })
}
  3. Recarga la configuración o reinicia Neovim.

Tips

  • Explora friendly-snippets para cientos de ejemplos.
  • Puedes crear snippets dinámicos usando Lua.
  • Lee la documentación oficial para trucos avanzados.

Ejemplo avanzado de snippet

return {
  s("for", {
    t("for "), i(1, "var"), t(" in "), i(2, "iterable"), t({" do", "  ", "  "}), i(0), t({"", "end"})
  })
}

Consejos de organización y productividad

  • Organiza tus snippets por lenguaje en subcarpetas dentro de nvim/lua/snippets/.
  • Adapta y reutiliza snippets de friendly-snippets para acelerar tu flujo.
