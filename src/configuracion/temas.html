Temas y personalización visual

¿Cómo cambiar de tema?

  • Pulsa Space + th para abrir el selector de temas.
  • Selecciona un tema y pulsa Enter.

Personalizar la UI

  • Edita nvim/lua/chadrc.lua y cambia la línea theme = "nombre_del_tema".
  • Guarda y reinicia Neovim para aplicar los cambios.

Ejemplo

M.ui = {
  theme = "onedark",
  theme_toggle = { "onedark", "one_light" },
}

Tips

  • Explora la lista de temas disponibles.
  • Puedes crear tus propios temas o modificar los existentes.
  • Usa :NvCheatsheet para ver atajos visuales.

Personalización avanzada con base46

  • Los temas de NvChad se gestionan desde el repositorio base46. Puedes crear o modificar temas editando los archivos en repos/base46/lua/base46/themes/.
  • Para crear un tema propio, copia un archivo de tema existente, renómbralo y ajusta los colores a tu gusto.
  • Recarga Neovim o selecciona tu nuevo tema desde el selector (Space + th).
  • Consulta la documentación en repos/base46/README.md para detalles y ejemplos avanzados.
