CSS Grid: El sistema de layout del futuro

¿Qué es CSS Grid?

CSS Grid es un sistema de layout bidimensional que permite crear diseños complejos de manera sencilla y flexible. A diferencia de Flexbox, que es unidimensional, Grid te permite controlar tanto filas como columnas simultáneamente.

Conceptos básicos

Container y Items

  • Grid Container: El elemento padre que contiene la propiedad display: grid
  • Grid Items: Los elementos hijos directos del container

Grid Lines

Las líneas que forman la estructura del grid, numeradas desde 1:

.grid-container {
    display: grid;
    grid-template-columns: 200px 1fr 200px;
    grid-template-rows: 100px 1fr 100px;
}

Propiedades principales

Grid Template Columns/Rows

Define el tamaño de las columnas y filas:

.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    grid-template-rows: 100px auto 100px;
}

Grid Gap

Espaciado entre elementos:

.grid {
    gap: 20px;
    /* Equivale a: */
    row-gap: 20px;
    column-gap: 20px;
}

Grid Areas

Permite crear layouts complejos con nombres:

.grid {
    display: grid;
    grid-template-areas: 
        "header header header"
        "sidebar main aside"
        "footer footer footer";
    grid-template-columns: 200px 1fr 200px;
    grid-template-rows: 80px 1fr 80px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

Ejemplo práctico: Layout de blog

<div class="blog-layout">
    <header class="header">Header</header>
    <nav class="sidebar">Sidebar</nav>
    <main class="content">Contenido principal</main>
    <aside class="widgets">Widgets</aside>
    <footer class="footer">Footer</footer>
</div>

.blog-layout {
    display: grid;
    grid-template-areas: 
        "header header header"
        "sidebar content widgets"
        "footer footer footer";
    grid-template-columns: 250px 1fr 300px;
    grid-template-rows: 80px 1fr 100px;
    min-height: 100vh;
    gap: 20px;
}

.header { grid-area: header; background: #333; color: white; }
.sidebar { grid-area: sidebar; background: #f5f5f5; }
.content { grid-area: content; background: white; }
.widgets { grid-area: widgets; background: #f0f0f0; }
.footer { grid-area: footer; background: #333; color: white; }

Responsive Grid

.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 20px;
}

Ventajas de CSS Grid

  1. Control bidimensional: Maneja filas y columnas simultáneamente
  2. Flexibilidad: Fácil de hacer responsive
  3. Semántica: El HTML puede ser semántico, el layout se maneja en CSS
  4. Potencia: Puede reemplazar frameworks de layout complejos

Conclusión

CSS Grid es una herramienta poderosa que ha revolucionado el diseño web. Su adopción es cada vez mayor y es esencial para cualquier desarrollador frontend moderno.
