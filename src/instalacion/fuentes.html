Instalación de Nerd Fonts

¿Por qué usar Nerd Fonts?

Las Nerd Fonts permiten que Neovim y sus plugins muestren iconos y símbolos correctamente, mejorando la experiencia visual.

Paso 1: Descargar una Nerd Font

  • Recomendado: JetBrainsMono Nerd Font
  • Descarga el archivo zip para tu sistema operativo.

Paso 2: Instalar la fuente

Linux

sudo apt install unzip
cd ~/Descargas
unzip JetBrainsMono.zip -d JetBrainsMono
mkdir -p ~/.local/share/fonts
mv ./JetBrainsMono/*.ttf ~/.local/share/fonts
fc-cache -f -v

Windows y macOS

  • Extrae el zip descargado.
  • Haz doble clic en los archivos .ttf y pulsa “Instalar”.

Paso 3: Configura la fuente en tu terminal

  • Abre la configuración de tu emulador de terminal.
  • Selecciona la fuente JetBrainsMono Nerd Font (o la que elegiste).
  • Guarda y reinicia el terminal.

¿Problemas?

  • Reinicia el terminal o la sesión.
  • Verifica que la fuente esté bien escrita y seleccionada.
  • Consulta la web oficial para más ayuda.
