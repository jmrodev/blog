Preparación y requisitos

Requisitos mínimos

  • Neovim 0.10.0 o superior
    nvim --version
  • Nerd Font (¡No usar la variante Mono!)
    Ejemplo: JetBrainsMono Nerd Font (no Mono)
  • Ripgrep (opcional, recomendado para búsquedas)
    rg --version
  • GCC y Make (para compilar plugins nativos)
    gcc --version y make --version

¿Cómo verificar los requisitos?

nvim --version
fc-list | grep Nerd
rg --version
gcc --version
make --version

¿Cómo instalar los requisitos?

sudo pacman -S neovim ripgrep gcc make # Manjaro/Arch
sudo apt install neovim ripgrep gcc make # Debian/Ubuntu

Backup y limpieza de instalaciones previas

Si ya usaste Neovim antes, haz backup y limpia:

mkdir ~/backup_nvim
cp -r ~/.config/nvim ~/.local/share/nvim ~/.cache/nvim ~/backup_nvim
rm -rf ~/.config/nvim ~/.local/share/nvim ~/.cache/nvim

Tips

  • Siempre haz backup antes de borrar.
  • Reinicia el terminal tras instalar fuentes o paquetes.
  • Consulta la guía oficial si tienes dudas.
